# Technoheight AI Static Website

This is a ready-to-upload static website. No coding tools are required.

## How to upload
1. Open the ZIP file.
2. Upload everything inside the folder to your repository.
3. Keep the folder structure exactly the same.
4. In GitHub Pages, publish from the root folder.

## Important
- Do not delete the assets folder.
- Do not rename service folders unless you also update links.
- Each folder with index.html creates a clean URL.

Examples:
/services/data-solutions/
/contact/
/blog/ai-outsourcing-operating-model/
